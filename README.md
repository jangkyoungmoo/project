# 이 레포지토리는 github 프로젝트를 진행하기 위한 공간입니다.

## 개발 환경
Framework: React(vite)
Language: JavaScript
Tool: VScode, Git, GitHub

## Branch
Main: 배포용 브랜치
Develop: 개발 중심 브랜치
feat: 기능 개발용 브랜치

## React(vite)
설치: npm install
실행: npm run dev

## Commit Messasge Convention
✨ Feat: 새로운 기능 추가
🐛 Fix: 버그 수정
📝 Docs: 문서 수정
🎨 Design: CSS스타일 수정

## 폴더 구조
'src': 실제 코드
'public': 정적 리소스
